# Efikasno pretraživanje proizvoda uz pomoć Bloom filtera

Ovaj projekat prikazuje implementaciju i eksperimentalnu analizu Bloom filtera - probabilističke strukture podataka koja omogućava brzo i memorijski efikasno provjeravanje pripadnosti elementa skupu.
